Introduction: Installation

To install thecord.py, use the package manager pip: pip install thecord Usage To use thecord.py in your project, you first need to import the library: import thecord You can then create a bot using the Client class: bot = thecord.Client() You can also use the Client class to define the bot's command prefix and token: bot = thecord.Client(command_prefix='!', token='YOUR_TOKEN_HERE')

Features

Slash command support: thecord.py allows users to interact with your bot through text commands, which can be defined using the @slash-command decorator. "Only You can see this message" support: thecord.py enables users to send private messages to the bot. Button Support: thecord.py allows users to interact with your bot through buttons. And more features are under development

Examples

import thecord bot = thecord.Client(command_prefix='!', token='YOUR_TOKEN_HERE') @bot.slash_command(name='test', description='test') def test(ctx): ctx.send("Hello from the mother ship.") bot.run() For more information and examples, please refer to the documentation in the GitHub repository.
